 ### Projects
 #### Project 1: Binary Classification – Breast Cancer Detection

Goal:
Build a model to classify tumors as malignant (cancerous) or benign (non-cancerous).

Key Steps:

Data cleaning and preprocessing
Feature selection
Training multiple models:
Logistic Regression
Random Forest
XGBoost
Neural Networks
Model evaluation using accuracy, precision, recall, F1-score

Outcome:
Compared multiple models and identified the best-performing classifier.

#### Project 2: Regression – Price Prediction

Goal:
Predict continuous values (e.g., house prices / product prices) using regression models.

Key Steps:

Data preprocessing (handling missing values, encoding, scaling)
Feature engineering to improve model performance
Training multiple models:
Linear Regression (baseline)
Random Forest Regressor
XGBoost Regressor
Neural Network
Hyperparameter tuning
Model evaluation using:
R² Score
Mean Absolute Error (MAE)
Mean Squared Error (MSE)

Outcome:
Built a robust regression pipeline and improved prediction accuracy using advanced models and feature engineering.
